# Kawaragi | Blog
Blog app reads posts from Markdown files.

## Getting started.
### Installation
Clone this repository
```bash
git clone https://github.com/triistam/kawaragi.ts
```

### Run locally:
```bash
deno run -A mod.ts
```

- Or deploy globally on https://deno.com/deploy
